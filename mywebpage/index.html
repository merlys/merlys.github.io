Like JQuery!

It is all about JavaScript and its famous library called JQuery. JQuery makes it possible to write less and do more. I added two cool features on this page, first feature changes the background color of this paragraph when hovered over. To see the second functionality, click on this paragraph.

I used the following code for the background color changing effect:

$("p").hover(function() {

$(this).css("background-color", "yellow");

}, function() {

$(this).css("background-color", "white");

});
